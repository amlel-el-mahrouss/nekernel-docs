\doxysubsection{Detailed Description}
Fork type, contains a data page. 

\begin{DoxyNote}{Note}
The way we store is way different than how other filesystems do, specific chunk of code are written into either the data fork or resource fork, the resource fork is reserved for file metadata. whereas the data fork is reserved for file data. 
\end{DoxyNote}
